Material 3 Floating Action Button — the most prominent, persistent action on a screen (usually one per screen).

```jsx
<Fab icon="add" ariaLabel="새로 만들기" />
<Fab extended icon="edit" label="작성하기" color="primary" />
<Fab size="large" icon="add" color="tertiary" />
```

Sizes: `small`, `regular` (default), `large`. Colors: `primary` (default), `secondary`, `tertiary`, `surface`. Set `extended` with a `label` for the pill form.
